# Introduction

Bush.js is a simple developer-friendly framework for building backend applications in Node.js.

It helps developers by providing:

- a quick way to scaffold a working project
- a clear file structure for app code
- built-in routing, controllers, and validation
- CLI tools for generating common files

## What Bush.js helps you build

- APIs for web and mobile apps
- backend services with authentication
- apps that need clear route and controller logic
- projects that benefit from generator-based scaffolding

## Why use Bush.js

- **Quick start:** Generate a new application with a single command.
- **Practical structure:** Know exactly where to add your code.
- **Less boilerplate:** Use generators to create controllers, requests, middleware, and routes.
- **Developer workflow:** Focus on features instead of framework setup.

## What you’ll learn in these docs

- how to scaffold a new Bush.js project
- how to run the project locally
- how to add routes, controllers, and middleware
- how to validate requests and secure endpoints
- how to configure the app for different environments
