# Basics

Basics covers complete application development using the framework project structure.

## What You Build in Basics

You build full backend features using:

- `app/` for business code
- `routes/` for transport entry points
- `config/` for runtime settings
- `database/` for schema lifecycle
- `scripts/` for project operations
- `src/` for app bootstrap/composition

## Application Flow

1. Route is matched
2. Middleware pipeline runs
3. Controller action executes
4. Validation/auth/policy checks run
5. Model and data operations complete
6. Response is returned
