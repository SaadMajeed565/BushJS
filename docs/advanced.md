# Advanced

Advanced covers direct core usage for custom architecture.

## What Advanced Solves

Use Advanced when you need:

- custom module/domain folder layout
- non-standard composition root
- monorepo/service integration
- direct framework API wiring without default conventions

## Core APIs Used in Advanced

- `Application` bootstrapping and lifecycle
- direct route and middleware registration
- direct auth provider and gate registration
- validation/model/schema integration in custom modules
- GraphQL and WebSocket composition in custom layouts

## Main Advanced Guide

- [Advanced Custom Architecture](advanced-custom-architecture.md)

## Expected Outcome

After completing Advanced, you should be able to design and run Bush.js apps with your own architecture standards while preserving framework capabilities.
